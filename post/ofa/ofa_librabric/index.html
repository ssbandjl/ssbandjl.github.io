OFA(开放Fabric联盟) - 利用最小CPU开销,为上层应用提供超高性能(线速),超低时延,最大带宽的开放组织

成员企业

OFA概览

OpenFabrics Alliance (OFA) 是一个基于开源的组织，负责开发、测试、许可、支持和分发 RDMA/Advanced Networks 软件以及 RDMA/Advanced Networks 软件的 OpenFabrics Enterprise Distribution。该联盟的使命是开发和推广软件，通过以最小的 CPU 开销直接向应用程序提供线速消息传递、超低延迟和最大带宽，从而实现最大的应用程序效率。

该联盟成立于 2004 年 6 月，当时名为 OpenIB 联盟，最初专注于开发独立于供应商、基于 Linux 的 InfiniBand 软件堆栈。2005 年，联盟承诺支持 Windows，此举将使软件堆栈真正跨平台。2006 年，该组织再次扩展其章程以包括对 iWARP 的支持，并在 2010 年增加了对 RoCE（RDMA over Converged Ethernet）的支持，以通过以太网提供高性能 RDMA 和内核旁路解决方案。2014 年，联盟再次扩大规模，成立了 OpenFabrics Interfaces 工作组，以调查和纳入对其他高性能网络的支持。

今天，OpenFabrics 联盟的愿景是为 RDMA 和内核旁路提供一个统一的、跨平台的、独立于传输的软件堆栈。传输独立性意味着用户可以利用相同的 RDMA 和内核旁路 API 在 InfiniBand、iWARP、RoCE 或其他结构上以不可知的方式运行他们的应用程序。为此，OFA 管理工具和开发资源，以编码、改进和发布基于标准的开源软件，这些软件具有支持关键任务应用程序所需的可扩展性和性能。联盟成员和其他贡献者参与工作组，以确保 RDMA/高级网络软件和 OFED 功能、易用性和部署以及与网络硬件和主要操作系统的互操作性。

该联盟通过举办年度开发人员研讨会来开发强大的生态系统，行业领导者可以在会上讨论 RDMA 结构的进步以及 IT 经理在部署下一代数据中心时面临的挑战。这些研讨会的成果推动了 RDMA/Advanced Networks Software 的进步，并且不仅仅是偶尔地推动了网络基础设施产品的改进。该联盟还通过我们的用户社区门户以及 IT 管理员可以共享信息、提出问题和搜索答案的地方为部署和管理 RDMA 结构的人员提供支持。

RDMA/高级网络解决方案广泛用于高性能计算和企业数据中心应用。在地平线上，OFA 正在寻求为云和超融合市场带来同样最大化的应用程序效率。RDMA/Advanced Networks 软件被推送到 kernel.org，并包含在 Red Hat、Novell、Oracle 和 Microsoft 的所有主要服务器操作系统中。RDMA/Advanced Networks 软件也被包括 Solaris 在内的其他操作系统所移植。OFA 不断促进行业对 RDMA、内核旁路和低延迟结构技术的认识和接受。

OpenFabrics 联盟欢迎 RDMA 架构的所有供应商和用户加入我们。如需更多信息，请访问会员页面

官方演示文稿

OFA简介-2021年9月

使命

网络对于现代高性能计算 (HPC) 系统的性能和可扩展性的爆炸式增长至关重要 从 2004 年开始，OFA 的章程旨在促进 InfiniBand 架构的快速采用，该架构已成为业界要求最苛刻的 HPC 平台的互连选择 在此基础上，OFA 正在扩大其使命，以期待它们将支持的新兴网络和应用程序，包括： • 机器学习 • 人工智能 • 云计算 • 企业数据中心

宣言

OpenFabrics Alliance (OFA) 的使命是加速先进Fabric的开发和采用，以造福先进的网络生态系统。 任务是通过以下方式完成的； 为开发和部署此类Fabric的人员创造合作机会，孵化和发展独立于供应商的Fabric开源软件，并支持和促进此类Fabric技术软件的使用

关于联盟

OFA 推动网络开源软件的开发，并支持和维护现有的Fabric技术

  • 支持Fabric软件的开发
  • 提供协作、交流和教育论坛以支持Fabric技术
  • 促进Fabric技术，包括此类技术的新兴用途
  • 提供支持Fabric技术的程序，例如培训、测试、验证
  • 促进与 SNIA(存储网络行业协会)、Linux 内核社区等相关组织和社区的合作

会员福利

OFA 促进开源架构社区中的开发人员、测试人员和最终用户之间的工作关系，以加速架构技术的开发和采用 OFA 的会员资格提供： • 成员是Fabric开源软件开发、OpenFabrics 互操作性测试和 OFA 推广活动的主要影响者 • 通过 OFA 合作伙伴联盟计划，与其他成员交流的机会以及接触更大的行业组织和思想领袖社区的机会 • 在贸易展、会议和研讨会上与 OFA 活动相结合的宣传和营销机会

成员层级

发起人 • 强烈影响 OFA 的外展、计划、合作伙伴关系、会议和营销 • 允许提名来自发起人成员组织的个人担任董事会官员和工作组主席/联合主席职位。 • 此级别包含采纳者和支持者级别的所有好处。 采用者 • 参与OFIWGA 提供的测试项目。 • 他们参与的任何工作组的投票成员。 • 此级别包含支持者级别的所有好处。 支持者 • 随时了解正在实施的外展、计划、合作伙伴关系、会议和营销工作。 • 在我们的网站上列出并在年度 OpenFabrics 研讨会上进行推广。 • 年度研讨会演讲者一张免费通行证。

工作组

▪ Fabric 软件开发平台工作组 (FSDPWG) • FSDP 履行 OFA 的使命，即加速先进结构的开发和采用，以造福于先进的网络生态系统。 ▪ 营销工作组 (MWG) • MWG 通过我们的成员、合作伙伴和 OpenFabrics 社区的创新促进结构的进步，确保 OFA 的战略得以实施。 MWG 还通过推动 OFA 年度研讨会、培训课程和在合作伙伴活动中提高 OFA 意识来促进社区之间的协作。 ▪ OpenFabrics 接口工作组 (OFIWG) • OFIWG 致力于 (1) 开发、测试和分发可扩展的开源框架，提供对高性能结构接口和服务的访问； (2) 可扩展的开源接口，符合 ULP 和应用程序对高性能结构服务的需求。 OpenFabrics 管理框架工作组 (OFMFWG) • OFMFWG 的章程是创建一个 OpenFabrics 管理框架，以开发、测试和分发可扩展的开源管理框架和接口

高级网络软件

Advanced Networks Software (ANS) 是一个综合概念，包含两个系列的高性能 API 和相关软件，用于当前和未来的 HPC、云和企业数据中心部署。 它专注于需要更高效网络、更快存储连接和可扩展并行计算的应用程序。 OpenFabrics 软件（verbs） OpenFabrics 接口 (libfabrics) 从 2004 年开始，ANS 使现代 HPC 系统的可扩展性和性能实现了爆炸式增长。 它准备对商业系统做同样的事情。

年度 OFA 研讨会

促进开发结构、部署结构、 并创建依赖结构的应用程序 ▪ 同类活动中面料开发人员和用户讨论新兴面料技术、就未来行业需求进行协作并解决当今存在的问题的唯一活动 ▪ 研讨会参与者包括： • 应用程序开发人员和最终用户 • 通信中间件开发商 • 网络和存储供应商和研究人员 • 操作系统解决方案开发商 • 企业数据中心经理和架构师 • 系统和网络管理员 • 系统原始设备制造商、架构师和集成商 • 内核开发人员和维护人员

OFA 行业联盟计划

客观的： • OFA 的行业联盟计划促进与 OFA 专家和工作组的行业合作，共同推进织物技术 ▪ 联盟优势： • 提供一个平台来孵化和发展供应商独立的架构开源软件 • 推动技术交付，使组织和整个行业都受益 • 提供联合营销计划以支持技术交付 • OFA 研讨会参与 • 包含在 OFA 成员通讯中 • 提高行业意识的教育会议 • 促进创新技术和应用的采用
